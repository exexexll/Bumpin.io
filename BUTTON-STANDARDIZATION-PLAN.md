BUTTON DESIGN STANDARDIZATION
==============================

## Global Brand Color

Primary: #ffc46a (golden yellow)
Usage: Main CTAs, primary actions, brand elements

## Current Button Inconsistencies

Found buttons using:
- bg-green-500 (various success actions)
- bg-blue-500 (info/secondary actions)
- bg-purple-500 (admin/special features)
- bg-red-500 (destructive/cancel actions)
- bg-white/10 (secondary/ghost buttons)

## Standardization Rules

### Primary Actions (Use #ffc46a):
- Submit forms
- Continue/Next steps
- Upgrade account
- Save settings
- Send invites
- Start calls

### Secondary Actions (Keep as-is):
- Cancel buttons (bg-white/10)
- Back buttons (bg-white/10)

### Destructive Actions (Keep red):
- Delete account
- End call
- Report user
- Block user

### System Messages (Keep semantic colors):
- Success: green-500
- Error: red-500
- Warning: yellow-500
- Info: blue-500
